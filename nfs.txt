Create server:
1- sudo apt-get install nfs-kernel-server
2- sudo mkdir cloud
3- sudo chown nobody:nogroup cloud
4- Adicione no arquivo: /etc/exports -> /home/xandao/cloud *(rw,sync,no_root_squash,no_subtree_check)
5- sudo exportfs -a
6- sudo /etc/init.d/nfs-kernel-server start

Create client:
1- sudo apt-get install nfs-common
2- sudo mkdir cloud
3- sudo mount 192.168.1.100:/home/xandao/cloud /home/xandao/cloud
