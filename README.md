# Sistemas distrbuídos

Sistemas distrbuídos - UTFPR - CM - (2017-2)

### Compilar

```
$ make compile

```

### Executar localmente

```
$ make execute NP="value > 0" FILE="PATH to file JSON"

```

### Executar remotamente

```
$ make execute NP="value > 0" IP="IP_SLAVE" FILE="PATH to file JSON"

```
